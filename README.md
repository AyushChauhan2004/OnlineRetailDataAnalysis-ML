# 📊 Online Retail Analytics: EDA, Statistics & Machine Learning

## 🚀 Overview

This project performs an end-to-end analysis of an online retail dataset to extract meaningful insights and build predictive models.
It combines **Exploratory Data Analysis (EDA)**, **statistical hypothesis testing**, and **machine learning** to understand sales behavior and predict revenue.

---

## 🎯 Objectives

* Analyze sales trends and customer purchasing behavior
* Identify key factors influencing revenue
* Perform statistical validation using hypothesis testing
* Build and compare machine learning models
* Classify high-value vs low-value transactions

---

## 🛠️ Tech Stack

* **Languages:** Python
* **Libraries:** Pandas, NumPy, Matplotlib, Seaborn, Scikit-learn
* **Concepts:** EDA, Feature Engineering, Regression, Classification, Hypothesis Testing

---

## 📊 Exploratory Data Analysis

Key visualizations performed:

* Revenue distribution (normal & log scale)
* Sales by country
* Transactions by hour and month
* Correlation heatmap
* Outlier detection using boxplots
* Quantity vs revenue relationship

---

## 📐 Statistical Analysis

Performed hypothesis testing using t-test:

* **Null Hypothesis (H₀):** No difference between weekday and weekend sales
* **Result:** Rejected H₀ (p-value < 0.05)
* **Conclusion:** Significant difference exists between weekday and weekend revenue

---

## 🤖 Machine Learning Models

### 🔹 Regression Models

* Linear Regression
* Polynomial Regression
* Decision Tree Regressor

📌 **Best Model:** Selected based on R² score and RMSE

---

### 🔹 Classification Models

* Logistic Regression
* Decision Tree Classifier

📌 Used to classify high-value vs low-value invoices

---

## 🌳 Model Interpretability

* Decision Tree visualization used for understanding feature impact
* Feature importance analysis performed

---

## 📈 Key Insights

* Revenue is strongly influenced by purchase quantity
* Certain countries contribute significantly more to total revenue
* Sales patterns vary across time (hour/month)
* Statistical testing confirms behavioral differences in sales
